# Plugin Lab

Use this workflow to try Codex plugins in a controlled way and turn useful experiments into reusable memory or templates.

## Experiments

| Plugin | First Experiment | Durable Output |
| --- | --- | --- |
| Documents | Generate a one-page project brief | `.docx` template or writing workflow |
| Spreadsheets | Build a project tracker workbook | `.xlsx` template and formulas |
| Presentations | Create a 5-slide strategy deck | PPTX template and outline workflow |
| PDF | Extract and summarize a PDF | Research-note workflow |
| Remotion | Create a short scripted video | Video script and component pattern |

## Experiment Rule

Each plugin experiment should produce one reusable artifact or one decision:

- reusable artifact: template, workflow, script, or example file
- decision: keep, revise, or discard the plugin workflow

## Review Questions

- Did this plugin save time compared with manual work?
- Is the output quality repeatable?
- What input format produces the best results?
- Should the workflow become a Skill, template, or automation?
