# People And Context

Use this file for non-sensitive collaboration context that helps future work. Do not store private personal details unless they are explicitly intended for durable use.

## Collaboration Defaults

- Ask fewer questions when a reasonable assumption is safe.
- Make assumptions explicit when they affect output quality.
- Keep reusable artifacts in the repository instead of leaving them only in chat.
- The user prefers practical Python learning in VS Code: demonstrate a working example, identify variable points, then assign a closely related exercise.
- The user prefers desktop utilities to open directly into the working UI from a desktop shortcut, with minimal intermediary windows.
